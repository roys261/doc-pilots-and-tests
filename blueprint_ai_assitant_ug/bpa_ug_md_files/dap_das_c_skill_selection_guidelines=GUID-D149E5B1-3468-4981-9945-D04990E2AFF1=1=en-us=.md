# Skill selection guidelines
The Blueprint AI Assistant skills catalog includes a range of
    categorized skills, each tailored to particular tasks. Select the skills that align with your
    intended workflow.
    
      Infrastructure skills
      Use infrastructure skills to:
      
        Set up foundational infrastructure components
        Configure networking, compute, or storage resources
        Establish base platform capabilities
      
    
    
      Platform skills
      Use platform skills to:
      
        Deploy application platforms
        Configure runtime environments
        Set up orchestration layers
      
    
    
      Security skills
      Use security skills to:
      
        Implement security controls
        Configure authentication and authorization
        Enforce compliance policies
      
    
    
      Monitoring skills
      Use monitoring skills to:
      
        Set up observability
        Configure alerting and logging
        Create dashboards for operations
      
    
    
      Integration skills
      Use integration skills to:
      
        Connect disparate systems
        Configure API gateways and event routing
        Set up data pipelines
