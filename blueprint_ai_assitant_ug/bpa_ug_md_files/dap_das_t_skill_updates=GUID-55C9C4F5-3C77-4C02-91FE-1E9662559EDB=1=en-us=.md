# Update the skills
Skills come bundled with the Blueprint AI Assistant releases and
		are updated automatically when you upgrade the Blueprint AI Assistant
		binary. There is no separate command to update individual skills. 
	  
		
				Download and install the latest Blueprint AI Assistant
					release. For more information, see Install
						Blueprint AI Assistant.
			 
		
				After the installation is complete, run the IDE integration again to apply the
					updated skills.
				
					For Devin Desktop: 
					bpa setup-ide windsurf
					For Claude code:
					bpa setup-ide claude-code
