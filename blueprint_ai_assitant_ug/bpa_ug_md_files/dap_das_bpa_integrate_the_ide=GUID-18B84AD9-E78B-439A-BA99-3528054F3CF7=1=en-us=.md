# Integrate the IDE
To enable agents to call the skills natively into your IDE, install the Blueprint AI Assistant skills. 
	  
		
				Install the skills for Devin Desktop:
				
					bpa setup-ide --target windsurf
				
			 
		
				Install the skills for Claude Code:
				
					bpa setup-ide --target claude-code
				
			 
	 
		It installs:
				Blueprint Assist skills for AI-powered blueprint authoring.
				Syntax highlighting and validation.
				Command completion and documentation.
				The bpa CLI within the integrated terminal of the IDE and agent
					environment.
