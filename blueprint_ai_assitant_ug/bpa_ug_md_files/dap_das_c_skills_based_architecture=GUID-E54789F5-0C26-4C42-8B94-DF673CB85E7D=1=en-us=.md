# Skills overview
Blueprint AI Assistant skills are modular, composable capabilities
    that map directly to Dell Automation Platform  plugins and
    operational workflows. 
    There is no separate skill invocation step. Users interact with skills through
      natural-language prompts in their IDE or using bpa CLI commands. Skills
      encapsulate specific infrastructure capabilities, enabling greater flexibility, reusability,
      and maintainability of blueprint deployments.
    
      Skill categories
      
        Plugin skills: One plugin skill is supported for each Dell Automation Platform plugin which provides the node-type knowledge, authoring
          guidance, and doc-lookup.
        Workflow skills: It guides specific Dell Automation Platform
          operational workflows.
        Authoring and scripting skills: It provides support for general blueprint
          development.
