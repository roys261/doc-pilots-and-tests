# Installation pre-requisites
Ensure that the components for your environment are ready before you begin to install
      Blueprint AI Assistant. 
    The following are the minimum system requirements that your environment must meet in order to
      proceed with installing Blueprint AI Assistant:
    
      Operating system: Windows 10/11, macOS 10.15+, or Linux: Ubuntu 20.04+, CentOS 7+, Red Hat
        Enterprise Linux 7+
      Memory: 8 GB RAM minimum, 16 GB recommended
      Storage: 100 GB of available disk space.
      Network: Internet connectivity for Dell Automation Platform API
        access and knowledge base updates.
      Certificates: Ensure that certificates are properly configured.
    
    Optional dependencies may be required based on your installation method and environment, such
      as Node.js 20 or later, which is only needed for source-based installations.
    The following are account requirements that you must have in order to proceed with Blueprint AI Assistant:
    
      Valid Blueprint AI Assistant
        account or organizational access
      Valid Dell Automation Studio license
      Access to Dell Automation Platform
      Appropriate permissions for target deployment platforms, such as: on-premises data center,
        and so on
      Access to internal blueprint repositories, if applicable
      Access from an IDE to an LLM to support the monitoring feature.
