# Install Blueprint AI Assistant on Linux or macOS
Complete the following steps to install and configure the tool in your Linux or macOS
		environment. 
		Ensure that you have met all pre-requisites
			before beginnging installation.  
		 
			
				Extract the archive.
				unzip dap-*.zip cd dap-*
			 
			
				Run the installer.
				./install.sh
				This installs the binary to ~/.local/bin and adds the
					directory to your PATH.
			
			
				Configure your environment.
				bpa setup
				
					Ensure that ~/.local/bin is in your
						PATH before running the above input. .
