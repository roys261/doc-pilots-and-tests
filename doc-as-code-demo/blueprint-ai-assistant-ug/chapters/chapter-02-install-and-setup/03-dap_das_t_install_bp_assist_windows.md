# Install Blueprint AI Assistant on Windows
Complete the following steps to install and configure the tool in your Windows
		environment. 
		Ensure that you have met all pre-requisites
			before beginning installation.  
		
			
				Obtain bpa-win-x64-v0.*-*-setup.exe from the Dell Automation Platform catalog.
			
			
				Right-click the installer and select Run as
						administrator. 
			
			
				Follow the installation wizard prompts and complete the installation.
				This installs bpa.exe and dap-bpa.exe to
						%USERPROFILE%\bin and adds the directory to your
					PATH.
				The knowledge base, skills, and rules are available in
						%USERPROFILE%\.blueprint-assist\.
			
			
				Open a new terminal and configure your environment.
				
					dap-bpa setup
